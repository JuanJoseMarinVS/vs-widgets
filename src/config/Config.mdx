# Inicialización de la librería

## 1. Autenticación en el registro privado

Crea un archivo `.npmrc` en la raíz de tu proyecto con el siguiente contenido para autenticarte y poder instalar la librería privada:

```bash
@<SCOPE>:registry=https://registry.npmjs.org/
//registry.npmjs.org/:_authToken=<tu-token-de-autenticación>
```

Recuerda reemplazar `TU_TOKEN_PERSONAL` con tu token de acceso personal de GitHub. Este token debe tener permisos para acceder al registro privado donde se aloja la librería.

## 2. Instalación de la librería

Instala la librería usando npm:

```bash
npm install @tecnologia-general/virtualsoft-widgets
```

## 3. Inicialización de la librería

La librería de widgets de VirtualSoft requiere una inicialización para poder personalizar los estilos y colores de los componentes. Esta inicialización se realiza mediante la función `init`, que acepta un objeto con variables CSS personalizadas y otras opciones.
